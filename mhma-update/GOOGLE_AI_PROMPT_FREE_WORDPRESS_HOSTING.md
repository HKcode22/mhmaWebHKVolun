# Google AI Prompt for Free WordPress Hosting Research

## Copy and paste this prompt into Google AI (or ChatGPT/Claude):

---

I need you to research and find completely FREE hosting solutions that can support a full WordPress installation with dynamic backend features, similar to what I have in LocalWP.

## CRITICAL REQUIREMENTS - The hosting MUST support ALL of these:

### WordPress Core Features:
- Full WordPress installation (self-hosted, not WordPress.com)
- PHP 8.0 or higher
- MySQL database (or MariaDB equivalent)
- WordPress REST API with full access
- WordPress admin panel access
- File uploads to WordPress media library via REST API
- Custom permalink structure

### Plugin Support (CRITICAL - MUST support these):
- JWT Authentication for WP REST API plugin
- Advanced Custom Fields (ACF) plugin
- Custom plugin installation and activation
- Ability to upload custom plugins via WordPress admin or FTP

### Theme Support:
- Custom theme installation and activation
- Ability to upload custom themes via WordPress admin or FTP
- Child theme support

### Database Access:
- phpMyAdmin or equivalent database management tool
- Direct database access for debugging
- Database backup/restore functionality

### File System Access:
- FTP/SFTP access to upload files
- File manager access via hosting control panel
- Ability to edit WordPress files (wp-config.php, etc.)

### API and Authentication:
- JWT token generation and validation
- Custom REST API endpoints
- CORS support for cross-origin requests
- API rate limiting configuration

### Performance and Reliability:
- No forced advertisements
- Custom domain support (not subdomain)
- SSL/HTTPS support (free Let's Encrypt)
- Daily backups
- Uptime guarantee (minimum 99%)
- Reasonable page load speeds

### Control Panel:
- cPanel, Plesk, or equivalent control panel
- One-click WordPress installation
- Easy site management

## WHAT I DON'T WANT:
- WordPress.com free plan (too limited)
- Static site hosting (GitHub Pages, Netlify, Vercel)
- Serverless-only hosting (Cloudflare Workers, AWS Lambda)
- Hosting with forced advertisements
- Hosting that limits plugins or themes
- Hosting that doesn't allow custom code

## RESEARCH FOCUS:
1. Identify ALL truly free WordPress hosting options that meet the requirements above
2. For each option, provide:
   - Hosting provider name
   - Free plan name
   - What's included (storage, bandwidth, databases)
   - What limitations exist (if any)
   - Whether it supports ALL the critical requirements listed above
   - Any hidden costs or upgrade requirements
   - User reviews/reputation
   - How to sign up

3. If NO free hosting exists that meets ALL requirements, clearly state this and provide:
   - The closest alternatives (what they support vs. what they don't)
   - The cheapest paid hosting options that DO meet all requirements
   - Cost breakdown for paid options

4. Include any creative workarounds or alternatives if traditional hosting doesn't work

## ADDITIONAL CONTEXT:
- I'm currently using LocalWP for local development
- I have a Next.js frontend that communicates with WordPress via REST API
- I need JWT authentication for API requests
- I use ACF for custom fields on program pages
- I need to upload images via the REST API
- I have no income/budget, so truly free is preferred
- If free is impossible, I need the cheapest option that meets ALL requirements

Please be thorough and realistic. Don't suggest options that don't meet the critical requirements. If nothing exists, be honest about that.
